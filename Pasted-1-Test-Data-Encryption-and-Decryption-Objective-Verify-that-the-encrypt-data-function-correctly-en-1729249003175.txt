1. Test Data Encryption and Decryption
Objective: Verify that the encrypt_data function correctly encrypts data and that the encrypted data can be decrypted back to the original.

Test Cases:
Test Case 1: Encrypt and decrypt a standard string.

Input: "Test data for encryption"
Expected Result: Decrypted data matches the original input.
Test Case 2: Encrypt and decrypt an empty string.

Input: ""
Expected Result: Decrypted data is an empty string.
Test Case 3: Encrypt and decrypt a string with special characters.

Input: "Special characters: !@#$%^&*()_+-=[]{}|;':\",./<>?"
Expected Result: Decrypted data matches the original input.
Test Case 4: Ensure that encrypted data is not the same as the original data.

Input: Any non-empty string.
Expected Result: Encrypted data differs from the original input.
